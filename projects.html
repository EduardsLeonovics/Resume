When to buy?

What is the best day of the month to invest in the stock market? Is it the start of the month? Maybe it is the 8th work day of each month? This project helped answer this question Technologies used:

• Yahoo API
• PostgreSQL
• Python
• Excel

As a stock market investor, I noticed that for some reason, when I receive my salaries (usually at the last days of the month) the stock market tends to be green, but long term investors like myself should avoid buying on these days, as buying securities on red days provides a higher yield on a 5-10 year basis. This project helped me understand which dates tend to be the weakest for the market.

Possible ways to improve the project further:

• Creating a Python executable, and launching all web scraping scripts through it by one click of a button. Making the whole report basically fully automated.
